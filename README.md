# Grand Dale Warriors

A static GitHub Pages browser card game set in Grand Dale. Open `index.html` directly or serve the folder with any static server.

## Features

- Deck Builder with multiple deck slots, collection counts, active deck selection, and minimum legal deck checks.
- Gacha page where gold buys random cards.
- Battle page with Warrior, Element Tool, Biome, and Magic card rules over a three-turn duel.
- Daily Reward page for one free random card per day.
- Daily Missions page with three rotating gold contracts.
- Lore page describing Grand Dale.
- Page-specific music from `assets/Music/`.
- Separate music and SFX volume controls.
- MIDI-style synthesized SFX patterns for player actions and duel events.
- Generated local art assets for the world, card sheets, biomes, victory, and defeat screens.
